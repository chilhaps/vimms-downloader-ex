# Vimm's Lair Downloader

## Description

This is a simple script to download all the files from Vimm's Lair. It uses the `requests` library to make HTTP requests and `BeautifulSoup` to parse HTML. The script downloads all the files from the specified page and saves them to a local directory.

## Requirements

- Python 3.x

## Installation

1. Clone the repository:

```bash
git clone repo_url
cd vimms-ownloader
pip install -r requirements.txt
python3 run.py
```
